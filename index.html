Terraform

A dive into the mind of a person learning how to set up Terraform on GCP for the first time
  1. Create GCP account, utilize 90 days of free tier stuff. I also created a new project called 'Terraform', though feel free to call it whatever you want, or even stick with the original project name given.
  2. Navigate to the 'Compute Engine' for the first time, which will result in loading time. Once complete. create a compute instance under 'Compute Engine'. Using Debian Linux 10 Buster. Picked LA region since it was close enough.
  3. Perform update/upgrade with apt-get
